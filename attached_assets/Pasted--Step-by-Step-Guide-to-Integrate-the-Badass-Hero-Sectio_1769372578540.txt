#### Step 4: Add JavaScript for Interactivity
Add this to `script.js` (or <script> tag). This handles button clicks, slider updates, and basic animations.

```javascript
document.addEventListener('DOMContentLoaded', () => {
  const raiseButton = document.querySelector('.raise');
  const raiseSlider = document.getElementById('raise-slider');
  const raiseValue = document.getElementById('raise-value');
  const raiseAmount = document.getElementById('raise-amount');

  // Toggle slider on Raise click
  raiseButton.addEventListener('click', () => {
    raiseSlider.style.display = raiseSlider.style.display === 'none' ? 'flex' : 'none';
  });

  // Update raise value display
  raiseAmount.addEventListener('input', () => {
    raiseValue.textContent = raiseAmount.value;
    raiseButton.textContent = `RAISE ${raiseAmount.value}`;
  });

  // Hover effects for abilities (scale up)
  document.querySelectorAll('.ability-card').forEach(card => {
    card.addEventListener('mouseenter', () => card.style.transform = 'scale(1.1)');
    card.addEventListener('mouseleave', () => card.style.transform = 'scale(1)');
  });

  // Low HP warning (example edge case)
  const hp = 57; // Fetch from game state
  if (hp < 20) document.querySelector('.hp-bar').classList.add('low-hp');
});
```

- **Integration**: Link <script src="script.js"></script> in HTML. For full game integration, tie buttons to your backend (e.g., via WebSocket for PVP).
- **Best Practices**: Use event delegation for performance. Debounce slider input if needed.
