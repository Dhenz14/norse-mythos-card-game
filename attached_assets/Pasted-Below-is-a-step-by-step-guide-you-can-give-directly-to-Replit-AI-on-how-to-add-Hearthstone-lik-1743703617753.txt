## 2. **Create a Reusable `NorseFrame` Component**
1. In your `src/components` folder, make a file called `NorseFrame.tsx` (or `.jsx` if you prefer JavaScript).  
2. This component will **wrap** your card’s content (art, text, stats) and apply **3D** styling for a Hearthstone-like feel.

**Example `NorseFrame.tsx`**:

```tsx
import React from 'react';
import './NorseFrame.css';

interface NorseFrameProps {
  children: React.ReactNode;
  frameUrl: string; // URL or path to your norse frame image
}

const NorseFrame: React.FC<NorseFrameProps> = ({ children, frameUrl }) => {
  return (
    <div className="norse-frame-container">
      <div className="norse-frame-3d">
        <img src={frameUrl} alt="Norse Frame" className="norse-frame-image" />
        <div className="norse-frame-content">
          {children}
        </div>
      </div>
    </div>
  );
};

export default NorseFrame;
```

**Notes**:  
- `frameUrl` references your Norse frame PNG/SVG.  
- `children` will be whatever content (card art, stats) you nest inside the frame.  
- We’ll apply 3D transforms via the `.norse-frame-3d` and `.norse-frame-image` classes in CSS.

---

## 3. **Add 3D Styling in `NorseFrame.css`**

1. **Perspective & 3D**  
   - Give the parent container a `perspective`, and transform the inner `.norse-frame-3d` for a slight tilt.  
2. **Hover Tilt**  
   - On hover, tilt or rotate the frame a bit more, simulating a 3D “pop.”

**Example `NorseFrame.css`**:

```css
.norse-frame-container {
  width: 200px;  /* adjust to your card size */
  height: 280px; /* maintain a Hearthstone-like aspect ratio */
  perspective: 800px; /* sets a 3D perspective */
  position: relative;
  margin: 10px;
}

/* This is the 3D "scene" container */
.norse-frame-3d {
  width: 100%;
  height: 100%;
  position: relative;
  transform-style: preserve-3d;
  transition: transform 0.3s ease;
}

/* Slight default rotation for a 3D look */
.norse-frame-3d {
  transform: rotateY(2deg) rotateX(1deg);
}

/* On hover, tilt more for emphasis */
.norse-frame-container:hover .norse-frame-3d {
  transform: rotateY(6deg) rotateX(3deg);
}

/* The frame image itself */
.norse-frame-image {
  width: 100%;
  height: 100%;
  object-fit: contain; /* or cover, depends on your frame design */
  position: absolute;
  top: 0;
  left: 0;
  pointer-events: none; 
  z-index: 2; /* sits above the card content */
}

/* Content area for card art, text, etc. */
.norse-frame-content {
  width: 100%;
  height: 100%;
  position: absolute;
  top: 0;
  left: 0;
  padding: 15px; /* space so content doesn't overlap the frame edges */
  z-index: 1;    /* behind the frame but above any background */
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
  transform: translateZ(1px); /* subtle push forward in 3D space */
}
```

**Key Points**:  
- `.norse-frame-container` sets **perspective** so child elements appear in 3D.  
- `.norse-frame-3d` uses **`transform-style: preserve-3d`** to maintain the 3D context.  
- **Hover** triggers a more pronounced rotation.  
- `.norse-frame-content` is placed behind the frame image but still in front of the background, with `translateZ(1px)` to separate it slightly in 3D space.

## 4. **Integrate `NorseFrame` into Your Card Rendering**
Wherever you render cards (e.g., `Card.tsx` or `Hand.tsx`), wrap the existing content in the new `NorseFrame`:

```tsx
import React from 'react';
import NorseFrame from './NorseFrame';

interface CardProps {
  name: string;
  mana: number;
  attack: number;
  health: number;
  artUrl: string;
  norseFrameUrl: string;
}

const Card: React.FC<CardProps> = ({
  name,
  mana,
  attack,
  health,
  artUrl,
  norseFrameUrl,
}) => {
  return (
    <NorseFrame frameUrl={norseFrameUrl}>
      {/* Existing content goes here */}
      <img src={artUrl} alt="Card Art" style={{ width: '100%', height: '60%' }} />
      <div style={{ fontWeight: 'bold', marginTop: '5px' }}>{name}</div>
      <div>Mana: {mana}</div>
      <div>Attack: {attack}</div>
      <div>Health: {health}</div>
    </NorseFrame>
  );
};

export default Card;
```

**Notes**:  
- Pass in `norseFrameUrl` for the custom Norse border.  
- The `Card` component’s children (art, stats, text) are rendered inside `.norse-frame-content` automatically.

## 5. **Optional: Dynamic Runic Glow or Embossing**
To amplify the **Norse** feel, add subtle glowing runes or an embossed look:

1. **CSS Filters**:  
   - Use `filter: drop-shadow(...)` or `box-shadow` in `.norse-frame-image` to mimic a glow effect around the edges.  
2. **Animated Rune Overlay**:  
   - A semi-transparent PNG overlay with glowing runes that slowly pulsate using CSS `@keyframes`.

**Example**:
```css
.norse-frame-image {
  /* existing rules */
  filter: drop-shadow(0 0 4px #00ffff); /* faint teal glow, for a “rune” effect */
}

@keyframes runeGlow {
  0% {
    opacity: 0.7;
  }
  50% {
    opacity: 1;
  }
  100% {
    opacity: 0.7;
  }
}

.glowing-runes {
  animation: runeGlow 2s infinite;
}
```
